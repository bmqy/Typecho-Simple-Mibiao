# Typecho-Simple-Mibiao

基于 Typecho 打造的简单米表系统。

功能简单，安装完成即可使用。

## 功能介绍

[x] 域名管理；
  - 支持设置所属平台；    
  - 支持设置出售页地址；    
  - 支持设置标签（暂未使用）；  
  
[x] 主题选择；  
  - 默认使用简单米表主题；  
  - 集成小票米表；

[x] 主题自定义设置；  
[x] 访问计数；
